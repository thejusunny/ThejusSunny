Juice engine is a 3D rendering engine that I have been working on as a personal project and also as part of self learning.

Project Details

Language/ Library - C++,DX11, HLSL

Status - On going


Features

  • Input layouts and pipeline
  • Vertex and Pixel shaders
  • Constant and Index buffers
  • 3D camera
  • Sprites and fonts
  • IMGUI intergration
  • Loading textures
